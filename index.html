erlusb by ndim

Erlang library for interfacing with USB devices (that is the goal, at least)

Eventually, erlusb wants to become the library to use when communicating with USB devices. Erlang's binary pattern matching should then make protocol implementation really easy.

Dependencies

Erlang, libusb (0.1 or 1.0, not decided yet)

Install

See included README file.
